Cần thiết lập Firebase

App cần kết nối Firestore để lưu dữ liệu. Làm theo các bước sau rồi mở lại file:

Bước 1 — Vào console.firebase.google.com → Tạo project
Bước 2 — Firestore Database → Tạo (chọn test mode)
Bước 3 — Project Settings → General → Your apps → Add web app → Copy firebaseConfig
Bước 4 — Mở index.html, tìm FIREBASE_CONFIG ở đầu script, dán config vào
Bước 5 — Tải lại trang
